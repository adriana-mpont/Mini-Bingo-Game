# Mini-Bingo-Game
This project is a Mini Bingo Game that will be a simple, terminal-based game built in **Python** where players can generate a random and valid bingo card and enjoy a smooth experience that follows understandable bingo rules. Our goal is to make it fun and easy for anyone to play.

# Organization
This project was organized by separating each part (core logic, UI design, tests and documents).  

The project follows a modular structure to keep the codebase clean and easy to maintain:
src/ – Core source code for the game.
game/ – Contains game logic such as card generation, number drawing, and win detection.
ui/ – Handles terminal display, menus, and notifications.
tests/ – Unit tests ensuring core mechanics work correctly.
docs/ – Project documentation, backlog, and sprint planning.
Root files – README.md, requirements.txt, and LICENSE.md for setup, dependencies, and usage terms.

## Running the projects and tests:

### Ensure you have Python 3 installed:
pip install python3

### Run the main program:
Mini-Bingo_Game/main.py
